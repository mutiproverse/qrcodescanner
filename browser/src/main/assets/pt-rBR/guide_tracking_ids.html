Não Rastreie

Há alguns anos, o W3C (World Wide Web Consortium) criou um mecanismo para os navegadores informarem aos servidores web que eles não gostariam de ser rastreados. Isso é feito incluindo um cabeçalho DNT (Do Not Track) com solicitações da web.

O cabeçalho DNT realmente não fornece nenhuma privacidade porque a maioria dos servidores da web o ignora. Por exemplo, Yahoo, Google, Microsoft e Facebook ignoram pelo menos alguns cabeçalhos DNT. A partir da versão 3.8, o Privacy Browser não tem mais a opção de enviar um cabeçalho DNT.

Modificação de URL

O navegador de privacidade remove IDs de rastreamento e redirecionamentos AMP (Accelerated Mobile Pages) das consultas de URL. Há uma postagem no blog atualizada com a lista atual de modificações. A modificação de URL pode ser desativada nas configurações se estiver causando problemas.

Cabeçalho X-Requested-With

O Google programou o WebView do Android para enviar um cabeçalho X-Requested-With a cada solicitação . O valor do cabeçalho X-Requested-With é definido para o ID do aplicativo, que no caso do Privacy Browser é com.redlable.privacybrowser.standard. Isso pode ser desativado usando as DevTools do WebView.
